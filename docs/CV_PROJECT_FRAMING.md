# CV Project Framing

## Project Title

Application Rationalization GenAI Agent

## One-Line Description

A LangGraph-based multi-agent system that automates enterprise application portfolio rationalization using LLM APIs, structured prompts, deterministic guardrails, and Excel/API reporting.

## Relevant Skills Demonstrated

- Python application engineering
- OpenAI-compatible LLM API integration
- LangGraph and LangChain orchestration
- Prompt engineering and structured response handling
- Deterministic validation around LLM outputs
- FastAPI REST service design
- Pydantic schema validation
- Excel data pipeline automation with pandas and openpyxl
- Evaluation reporting for GenAI reliability
- Docker packaging and GitHub Actions CI
- Git-based project hygiene with safe sample data

## Why The Project Is Not Overbuilt

The project avoids unrelated model training, Kubernetes, Databricks, Neo4j, and web search because they are not required for this use case. The included stack focuses on the real problem: reliable LLM-assisted decision automation over structured enterprise inventory data.

## Resume Bullets

- Built a LangGraph-based GenAI agent workflow to automate application portfolio ECR recommendations across L3 business capability clusters.
- Integrated OpenAI-compatible LLM APIs with structured prompts, response normalization, retry handling, and deterministic guardrails to improve output reliability.
- Added a production-style Python interface with FastAPI, Pydantic schemas, Docker packaging, CI tests, synthetic demo data, and evaluation reporting.
