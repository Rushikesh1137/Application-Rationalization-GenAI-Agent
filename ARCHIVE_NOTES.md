# Project Notes

This folder is a CV-ready version of the Application Rationalization GenAI Agent.

Included:

- LangGraph agent workflow under `src/`
- FastAPI service wrapper under `src/api.py`
- Pydantic schemas under `src/schemas.py`
- Synthetic demo input and output under `sample_data/` and `sample_output/`
- Evaluation script under `scripts/evaluate_output.py`
- Dockerfile and GitHub Actions workflow
- Documentation for setup, API usage, dry runs, output schema, and project framing

Excluded by design:

- Real client input workbooks
- Real client output files
- Local `.env`
- API keys
- Logs and traces
- Local virtual environments

The committed sample files are synthetic and are safe for a public personal GitHub repository.
